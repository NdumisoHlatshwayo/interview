Reflection on Digital Portfolio Development (STAR Method)
Situation

During this assignment, I was required to create a digital portfolio using GitHub. This included coding my CV using Markdown, recording and embedding a mock interview video, and publishing the entire portfolio using GitHub Pages. Initially, I had limited experience using GitHub for anything beyond basic repository creation, and I had never built a full portfolio or used Markdown in a professional way.

Task

My task was to develop a complete, professional digital portfolio that demonstrates my work readiness. This meant I needed to learn how to structure content using Markdown, present myself effectively through a CV, record a mock interview, and ensure that everything was accessible online through GitHub Pages. I also needed to reflect on my learning throughout the process.

Action

To complete the assignment, I first created my GitHub account using my student email and set up a new repository for my portfolio. I then learned the basics of Markdown, including headings, lists, and formatting, to properly structure my CV in a clean and readable way.

Next, I recorded a mock interview video where I answered common interview questions. I uploaded the video online and embedded it into my portfolio using Markdown/HTML.

After that, I configured GitHub Pages to publish my repository as a live website. I tested the link to ensure that all elements, including my CV and video, were displaying correctly.

Throughout the process, I made improvements to the layout and presentation of my portfolio to make it more professional and visually appealing. I also reflected on each part of the assignment using the STAR method to show my learning and development.

Result

As a result, I successfully created a fully functional digital portfolio that showcases my skills, experience, and readiness for the workplace. I gained practical experience using GitHub, Markdown, and GitHub Pages, which are valuable tools in the tech industry.

This assignment improved my ability to present myself professionally online and boosted my confidence in both technical and communication skills. I now understand how to create and manage a digital portfolio, and I feel more prepared for job opportunities in the future.
